# Release v1.0.1

## Titulo

CPSA Tests v1.0.1

## Descripcion

Version corregida de **CPSA Tests**, una aplicacion offline para practicar preguntas de estudio del CREST CPSA.

Incluye:

- APK Android compilado: `CPSA-Tests.apk`
- Version HTML original: `index_sin_descripciones.html`
- Logo incluido en el repositorio: `cpsa.png`
- App Android con nombre visible `CPSA Tests`
- Correccion para que la barra de notificaciones de Android no se sobreponga a la interfaz

La app funciona sin conexion a Internet y carga el quiz desde assets locales dentro de un WebView Android.

## Texto corto para el repositorio

Offline CREST CPSA practice quiz as a standalone HTML file and compiled Android APK.
